Feels Bad Man Super Computer

Overview

We own one of the sweetest super computers. With 1024 cores and 1024 GB of RAM, this computer is blazin fast. We even went as far as to load Windows Server 2003 onto it. Feels Bad Man is currently hosting this website! Amoung running some of the scripts that take a long time to run, Feels Bad Man is also working on cracking SHA-2.

Hardware

Here is a quick overview of Feels Bad Man's hardware. There are 512 nodes.

  • 512 blazin Intel Core2 Duo L7400 @ 1.50GHz
  • 1024 agile 1 GB sticks of DDR2 RAM

Each node is connected to each other through an accelerated 1/10 coaxial-based 10Base2 network card.
